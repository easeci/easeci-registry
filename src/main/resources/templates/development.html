Each software developer with some Java programming language knowledge can add own Performer to this repository. Our validator not allow to upload malformed file, malware or some other wrong things. If you want to add your own, custom Performer, you need to comply with strict standards of code contract. Main rules:
- The Performer script must contains of only one file ended with '.jar' extension
- JAR entry class must extends class and implement abstract methods specified in extension-api module
- Example Performer is here: github repository
Look at documentation for more details.
