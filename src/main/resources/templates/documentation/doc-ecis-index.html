ECIS-1

The document describes the operation and format of the Easefile file. There we will find a syntax file from which you can create a pipeline. It is a basic document that provides information especially for a user who wants to start working with EaseCI.

ECIS-2

ECIS-2 został stworzony do określenia spójnego standardu, zwartego interfejsu, przez który EaseCI Core może dogodnie i sprawnie komunikować się z EaseCI Extension. ECIS-2 określa sposób w jaki deweloper powinien implementować Performer czyli plugin typu DIRECTIVE_PLUGIN.
