Jak zainstalować extension w instancji EaseCI Core?

Wystarczy, że otworzysz plik w swoim workspace o nazwie plugins.yml
Do właściwości extensions dodajemy nowy element listy: 

extensions:
  - plugin:
      name: artifactory
      version: 1.0.0

Możemy dodawać dowolną listę extensions dowolnych rodzajów. W celu sprawdzenia, jakie mamy zainstalowane rozszerzenia w naszej instancji, użyj endpointu REST API: GET {easeci_core_address}/plugin/directives
