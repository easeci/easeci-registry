CommandParameter

io.easeci.extension.command.CommandParameter

Obiekt, który reprezentuje wartość, którą możemy przekazać wywołując dyrektywę. $git clone branch:master w tym przykładzie to branch:master jest użyciem parametru komendy.

Field name Field Type Field Description
parameterName String Nazwa parametru
parameterValue String Wartość parametru
parameterType VariableType Typ parametru
