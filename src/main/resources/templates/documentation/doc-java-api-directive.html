Directive

io.easeci.extension.command.Directive

Method name Method Description

getDirectiveName()

Method description Ta metoda musi zwracać nazwę dyrektywy. Jeżeli w Easefile chcemy wywoływać dyrektywę za pomocą $git, to metoda musi zwracać wartość: git lub $git

Method arguments -

Method returns Metoda zwraca nazwę dyrektywy

getAvailableCommandList()

Method description Metoda zwraca wszystkie dostępne komendy jakie użytkownik Performera będzie mógł użyć w deklaracji Pipeline'u w Easefile. Dostępną komendę, możemy użyć tak: $git clone. W tym przykładzie to właśnie clone jest jedną z command.

Method arguments -

Method returns Returns list of Command[] objects

provideCode(String parameters)

Method description Jest to kluczowa metoda tego interfejsu. Kiedy użytkownik użyje tą dyrektywę, EaseCI Core wywoła tą metodę w celu pozyskania kawałka kodu w języku skryptowym np. Python, który będzie dalej wykonany w procesie wykonywania Pipeline'u.
