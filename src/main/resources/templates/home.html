What is Ease CI ?

EaseCI is a tool as a network application that enables easy and quick deployment.
Simplicity and minimalism are the main features of EaseCI.

About Ease CI Registry

The EaseCI registry acts as an extension repository (or plugins, if you prefer). In EaseCI, each extension is called Performer or just Plugin. The Performer is a JAR archive that fulfills the agreed contract (more in the "For developers" section). Each plug-in is ready for manual or automatic delivery of a special entry in the Performers configuration. Thanks to this, your EaseCI Core instance can automatically download extensions at startup, which allows you to maintain the development environment in the "Infrastructure as a code" model. Each JAR downloaded from EaseCI must meets security standards and is subjected to tests and validation filters before adding. Each developer can start working on the Performer and at their own discretion expand EaseCI the way he likes.
