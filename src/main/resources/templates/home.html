About Ease CI Registry

The EaseCI registry acts as an extension repository (or plugins, if you prefer). In EaseCI, each extension is called Performer. The Performer is a Python script that fulfills the agreed contract (more in the "For developers" section). Each plug-in is ready for manual or automatic delivery of a special entry in the Performers configuration. Thanks to this, your EaseCI Core instance can automatically download extensions at startup, which allows you to maintain the development environment in the "Infrastructure as a code" model. Each script downloaded from EaseCI meets security standards and is subjected to tests and validation filters before adding. Each developer can start working on the Performer and at their own discretion expand EaseCI the way he likes.
